 E-Commerce Sales Dashboard (Power BI)
 Project Overview

This project presents an interactive and data-driven E-Commerce Sales Dashboard built using Microsoft Power BI.
The main goal is to analyze and visualize the performance of an e-commerce business, transforming raw sales data into valuable insights for data-driven decision-making.

Key Features:

Sales performance by product category, sub-category, and region

Year-over-year trend analysis for sales and profit

Profit margin and revenue insights using DAX measures

Customer and order growth visualization

Interactive filters and slicers for detailed exploration

Data cleaning and transformation using Power Query

Skills & Tools Used:

Power BI | DAX | Power Query | Data Modeling | Data Visualization | E-Commerce Analytics | Business Intelligence

Objectives:

Track overall sales performance and profitability in real time

Identify customer trends and high-performing product categories

Support better business decisions through data visualization

Demonstrate analytical and dashboard-building skills using Power BI
